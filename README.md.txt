# Movie Revenue Prediction

## Objective
Build a machine learning model to predict movie box office revenue before release, using features such as budget, cast, genre, social media buzz, and trailer statistics.

## Project Workflow
1. Data Collection
2. Data Cleaning and Exploration
3. Feature Engineering
4. Model Building (Regression)
5. Model Evaluation
6. Deployment (Interactive Website)
7. Project Documentation

## Folder Structure
- `data/` : Dataset files
- `notebooks/` : Jupyter notebooks for experiments
- `src/` : Python scripts and utility code
- `web/` : Frontend web application
- `docs/` : Documentation, diagrams
